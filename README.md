# Machine Learning Internship Projects

## 📌 Introduction
This repository contains all the tasks completed during my Machine Learning Internship. The goal of these projects is to apply data analysis, visualization, and machine learning techniques on real-world datasets.

---

## 📊 Tasks Included

### 🔹 Task 1: Iris Dataset Visualization
- Loaded dataset using pandas
- Explored data using shape, head, and columns
- Created scatter plots, histograms, and box plots

### 🔹 Task 2: Credit Risk Prediction
- Handled missing values
- Performed data visualization (EDA)
- Trained classification model (Logistic Regression / Decision Tree)
- Evaluated model using accuracy and confusion matrix

### 🔹 Task 3: Customer Churn Prediction
- Cleaned dataset and removed unnecessary columns
- Encoded categorical variables (Gender, Geography)
- Trained model to predict churn
- Analyzed feature importance
